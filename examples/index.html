date-range-slider

Interactive date range picker built on noUiSlider

Default Configuration

Just pass an element — gets 3-year range, all controls enabled.

import createDateRangeSlider from 'date-range-slider';
import 'date-range-slider/dist/date-range-slider.css';

const picker = createDateRangeSlider('#my-element', {
  onChange: ({ start, end, days }) => {
    console.log(`${start} — ${end} (${days} days)`);
  }
});

Custom Date Range & Presets

1-year range with custom preset buttons, no zoom controls.

createDateRangeSlider('#el', {
  minDate: new Date(2025, 0, 1),
  maxDate: new Date(2025, 11, 31),
  startDate: new Date(2025, 3, 1),
  endDate: new Date(2025, 6, 1),
  showZoom: false,
  presets: [
    { label: 'Q1', days: 90 },
    { label: 'Q2', days: 180 },
    { label: 'H1', days: 182 },
    { label: 'Full Year', days: 'max' },
  ],
});

Minimal — Slider Only

No presets, no zoom, no nudge — just the slider and calendar.

createDateRangeSlider('#el', {
  showPresets: false,
  showZoom: false,
  showNudge: false,
});

Programmatic API

const picker = createDateRangeSlider('#el');

// Get current selection
const { start, end, days } = picker.getRange();

// Set range programmatically
picker.setRange(new Date(2024, 0, 1), new Date(2024, 5, 30));

// Clean up
picker.destroy();
